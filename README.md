# 🎬 Bollywood Movie Success Prediction

A Machine Learning-powered web application that predicts whether a Bollywood movie is likely to become a **Hit** or **Flop** based on its financial performance indicators.

## 🚀 Project Overview

This project uses multiple Machine Learning classification algorithms to analyze movie-related financial data and predict box office success.

The application provides a modern web interface where users can enter movie statistics and receive instant predictions using different ML models.

* 🎬 Predict movie success (Hit / Flop)

## 🧠 Machine Learning Models Used

| Model               | Accuracy |
| ------------------- | -------- |
| KNN                 | 89.36%   |
| Random Forest       | 84.57%   |
| Decision Tree       | 84.04%   |
| Logistic Regression | 72.87%   |
| SVM                 | 71.28%   |

🏆 Best Performing Model: **KNN**
