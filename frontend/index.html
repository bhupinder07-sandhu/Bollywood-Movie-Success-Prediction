About Project

This machine learning application predicts the success of a Bollywood movie using multiple classification algorithms. The system compares KNN, Random Forest, Decision Tree, Logistic Regression, and SVM models to determine whether a movie is likely to become a Hit or a Flop. The best-performing model is automatically identified based on accuracy and performance evaluation.
